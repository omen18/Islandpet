## вңҰ What is IslandPet?

Most productivity apps treat you like a machine вҖ” track hours, hit targets, move on. **IslandPet flips that entirely.**

At its core, IslandPet is a **virtual companion system** woven into your iPhone's Dynamic Island. Every Pomodoro session you complete feeds your pet вҖ” giving it XP, unlocking new traits, triggering evolutions. Skip sessions and your companion gets sleepy. Stay consistent and it thrives.

It's not gamification for the sake of it. It's **emotional design as productivity infrastructure.**

```
Focus Session вҶ’ XP Earned вҶ’ Pet Reacts вҶ’ Streak Grows вҶ’ Evolution Unlocked
     вҶ‘                                                           вҶ“
     в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ You come back again в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

## вңҰ Core Features

<br/>

### рҹ§  Focus Engine

| Feature | Details |

| **XP System** | Every minute of focus = earned growth |

<br/>

### рҹҗҫ The Pet System

Your companion isn't static вҖ” it's a **living state machine** with memory.

```
States:         рҹҳ„ Happy  вҶ’  рҹҳҙ Sleepy  вҶ’  рҹ’Ө Idle  вҶ’  вңЁ Evolving
Triggers:       Focus     вҶ’  Rest       вҶ’  Neglect  вҶ’  Milestone
Personality:    Trait-based, deterministic, consistent across sessions
Memory:         Responds differently based on your history with it
```

No AI black boxes. No randomness. Just a **well-designed personality engine** that feels alive because it was *designed* to feel alive.

### рҹ“Қ Dynamic Island Integration

This is where IslandPet gets wild.

> **Your pet doesn't just live in the app вҖ” it lives in your phone.**

During focus sessions, your companion takes up residence in the Dynamic Island:

- рҹҹў **Compact view** вҖ” pet animation + session timer
- рҹ”” **Notification view** вҖ” pet reaction to your action
- рҹ“І **Expanded view** вҖ” full session controls + pet mood
- вҸ№пёҸ **Quick stop** вҖ” tap to end session right from the island

Built with **ActivityKit** + **WidgetKit**. Real system-level integration, not a workaround.

<br/>

### рҹҺ¬ Evolution Cinematic

When your pet evolves, it doesn't just level up silently.

It gets a **full-screen cinematic moment:**

- Particle burst effects
- Layered motion animations
- Haptic feedback choreography
- **Shareable 1080Г—1920 story-ready export**

Because milestones should *feel* like milestones.

## вңҰ Getting Started

### Prerequisites

```bash

```

### Installation

```bash

```
